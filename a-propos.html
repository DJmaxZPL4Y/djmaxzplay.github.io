Je me présente

DJmaxZPL4Y sous le vrai nom de ZUMSTEIN Maxime, je suis un jeune développeur passioné de 16 ans. Ayant commencé à apprendre le HTML ainsi que le CSS lorsque j'avais seulement une douzaine d'année je me suis orienté vers le language JAVA en créant mes premiers plugins sur le célèbre jeu composé uniquement de cubes intitulé Minecraft. Depuis ce moment j'ai découvert une vraie passion pour l'informatique et plus particulièrement le développement sous JAVA. Je décide donc de mettre mais connaissances sur internet afin de vous proposer de créer ce dont vous avez besoin. Toutes mes créations sont gratuites puisque je les fais dans le but de m'améliorer. Alors n'hésitez pas à passer sur la page Mes services si vous avez besoin d'un logiciel, de plugins Minecraft ou même d'un site web.
